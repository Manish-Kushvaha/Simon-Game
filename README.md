# 🎮 Simon Game

A fun and interactive memory-based game built using HTML, CSS, and JavaScript. Test your memory skills by repeating the sequence of colors generated by the game.

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript

## 🧠 How the Game Works

- The game starts when you press any key
- A random color sequence is generated
- You must repeat the sequence by clicking the buttons
- Each level adds a new step to the sequence
- If you click the wrong color, the game ends

---

## ✨ Features

- 🎯 Interactive gameplay
- 🔊 Sound effects for each button
- 📈 Increasing difficulty levels
- ❌ Game over detection
- 🔁 Restart functionality
- 🎨 Clean and responsive UI
